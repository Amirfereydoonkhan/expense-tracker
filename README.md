# Expense Tracker — Personal Finance Web App

A browser-based expense tracking application that allows users to log 
income and expenses, calculate balances in real time, and persist data 
across sessions using the browser's LocalStorage API.

## 🔗 Live Demo
[View Live App](https://amirfereydoonkhan.github.io/expense-tracker/)

## 🛠 Tech Stack
- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage API

## ✨ Features
- Log income and expense transactions
- Real-time balance calculation
- Transaction history display
- Data persistence using LocalStorage
- Clean, intuitive UI for non-technical users

## 🚀 Getting Started

Clone the repository:
git clone https://github.com/Amirfereydoonkhan/expense-tracker.git

Open index.html directly in your browser — no install needed.
